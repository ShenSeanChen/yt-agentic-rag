Agentic RAG

This combines RAG (Retrieval-Augmented Generation) with Tool Calling to create an AI that can both answer questions AND take actions.

How It Works

1. Retrieve relevant context from knowledge base

2. Decide if tools are needed

3. Execute tools (calendar, email)

4. Generate response with citations
